Download each installation file of game


  1. Go to drive.google.com.

  2. Go to Shared with me.


  3. Right-click each (.rar) file and the click Download.


  4. Select DOWNLOAD ANYWAY if an alert says that Google Drive can’t scan file for viruses. A file can be over 100 MB which is the limit for scanning.



Extract multiple (.rar) files with WinRAR


  1. Select all the (.rar) files.


  2. Right-click and select Extract files...



  3. Select OK.

  4. Enter password if required and select OK.


If you don’t have WinRAR, go here to get and to install WinRAR.



Install game


  1. Go inside the folder that you have extracted.



  2. Double-click an (.exe) file and finish the installation.



  3. Repeat the same for another (.exe) file.



The End


Muat turun setiap fail pemasangan permainan


  1. Pergi ke drive.google.com.

  2. Pergi ke Shared with me.


  3. Klik kanan setiap fail (.rar) dan klik Download.


  4. Pilih DOWNLOAD ANYWAY jika amaran mengatakan bahawa Google Drive tidak boleh mengimbas fail untuk virus. Fail boleh melebihi 100 MB iaitu batas pengimbasan.

Ekstrak fail-fail (.rar) dengan WinRAR


  1. Pilih semua fail (.rar).



  2. Klik kanan dan pilih Extract files...



  3. Pilih OK.

  4. Masukkan kata laluan jika perlu dan pilih OK.


Jika anda tidak ada WinRAR, pergi ke sini untuk mendapatkan dan memasang WinRAR.


Pasang permainan


  1. Pergi ke dalam folder yang anda telah ekstrak.



  2. Klik dua kali pada fail (.exe) dan selesaikan pemasangan.



  3. Ulang perkara yang sama untuk fail (.exe) yang lain.
