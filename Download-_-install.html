Extract multiple (.rar) files with WinRAR



  1. Select all the (.rar) files.

  2. Right-click and select Extract files...

  3. Select OK.

  4. Enter password if required and select OK.

Muat turun setiap fail pemasangan permainan


  1. Pergi ke drive.google.com

  2. Pergi ke Shared with me.

  3. Klik kanan setiap fail (.rar) dan klik Download.


Ekstrak fail-fail (.rar) dengan WinRAR


  1. Pilih semua fail (.rar).

  2. Klik kanan dan pilih Extract files...

  3. Pilih OK.

  4. Masukkan kata laluan jika perlu dan pilih OK.


Jika anda tidak ada WinRAR, pergi ke Download WinRAR untuk mendapatkan dan Memasang WinRAR.


Pasang permainan


  1. Pergi ke dalam folder yang anda telah ekstrak.

  2. Klik dua kali pada fail (.exe) dan selesaikan pemasangan.

  3. Ulang perkara yang sama untuk fail (.exe) yang lain.
